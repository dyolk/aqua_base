# Aqua_base 统一 UID_GID 分配表

> 所有从本项目构建的镜像必须使用如下 UID/GID，宿主机挂载文件/目录的属主必须与之对齐。

|  应用名  |  UID  |  GID  |     说明     |
|:-----:|:-----:|:-----:|:----------:|
| nginx | 10000 | 10000 | nginx 镜像使用 |

## 使用方法

- 禁止在容器内使用 `root` 用户
- 挂载宿主机文件/目录时请执行以下命令：
  ```bash
  sudo chown -R <UID>:<GID> /host/path
  ```
